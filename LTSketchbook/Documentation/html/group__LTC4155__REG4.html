Detailed Description

REG4 Register.

7 6 5 4 3 2 1 0
EXT_PWR_GOOD USBSNS_GOOD WALLSNS_GOOD AT_INPUT_ILIM INPUT_UVCL_ACTIVE OVP_ACTIVE OTG_FAULT BAD_CELL
  • CommandCode: 0x04
  • Contains Bit Fields:
    • EXT_PWR_GOOD : External power (Wall or USB) available.
    • USBSNS_GOOD : USB input voltage valid.
    • WALLSNS_GOOD : WALL input voltage valid.
    • AT_INPUT_ILIM : Input current limit control circuit engaged.
    • INPUT_UVCL_ACTIVE : Input undervoltage limit control circuit engaged (brownout).
    • OVP_ACTIVE : Overvoltage protection fault.
    • OTG_FAULT : USB On-The-Go step-up regulator fault shutdown.
    • BAD_CELL : Battery unresponsive to charging for 30 minutes.
