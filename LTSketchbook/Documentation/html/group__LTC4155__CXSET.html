Detailed Description

CXSET Bit Field.

End-of-Charge indication current threshold setting. This threshold can also be used for charge termination if TIMER is set to 8_Hour_or_CoverX.

  • Register: REG2
  • CommandCode: 0x02
  • Size: 2
  • Offset: 0
  • MSB: 1
  • MASK: 0x03
  • Access: R/W
  • Default: 0
