Detailed Description

WALLILIM Bit Field.

WALL total input current limit setting. Register contents will be modified by the LTC4155 upon certain events. See the datasheet section Input Current Regulation.

  • Register: REG1
  • CommandCode: 0x01
  • Size: 5
  • Offset: 0
  • MSB: 4
  • MASK: 0x1F
  • Access: R/W
  • Default: n/a
