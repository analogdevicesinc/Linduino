Detailed Description

REG1 Register.

7 6:5 4:0
PRIORITY TIMER WALLILIM
  • CommandCode: 0x01
  • Contains Bit Fields:
    • PRIORITY : Input connector priority swap setting. Setting PRIORITY will cause power to be drawn from USB when both WALL and USB power are present.
    • TIMER : User Programmed Li-Ion charger safety timer setting. This internal timer begins at the onset of constant-voltage charging. Charging terminates at the expiration of the selected time.
    • WALLILIM : WALL total input current limit setting. Register contents will be modified by the LTC4155 upon certain events. See the datasheet section Input Current Regulation.
