Detailed Description

TIMER Bit Field.

User Programmed Li-Ion charger safety timer setting. This internal timer begins at the onset of constant-voltage charging. Charging terminates at the expiration of the selected time.

  • Register: REG1
  • CommandCode: 0x01
  • Size: 2
  • Offset: 5
  • MSB: 6
  • MASK: 0x60
  • Access: R/W
  • Default: 0
