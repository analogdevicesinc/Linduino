Detailed Description

LOCKOUT_ID_PIN Bit Field.

Disables autonomous USB OTG operation when the ID pin is grounded by an appropriate USB cable.

  • Register: REG0
  • CommandCode: 0x00
  • Size: 1
  • Offset: 5
  • MSB: 5
  • MASK: 0x20
  • Access: R/W
  • Default: 0
