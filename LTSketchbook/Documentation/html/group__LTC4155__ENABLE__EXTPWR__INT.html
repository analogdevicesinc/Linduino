ENABLE_EXTPWR_INT Bit Field.

Enable USBSNS_GOOD, WALLSNS_GOOD, EXT_PWR_GOOD external power available interrupts.

  • Register: REG6
  • CommandCode: 0x06
  • Size: 1
  • Offset: 5
  • MSB: 5
  • MASK: 0x20
  • Access: R/W
  • Default: 0
