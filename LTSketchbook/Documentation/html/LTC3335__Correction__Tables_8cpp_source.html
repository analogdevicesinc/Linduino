1 /*!
2 LTC3335: Nanopower Buck-Boost DC/DC with Integrated Coulomb Counter
3 
4 Graphs G37 - G44 in the LTC3335 datasheet show the typical error in the
5 coulomb counter and battery current measurement error for various IPEAK,
6 VOUT, and battery voltage conditions. The LTC3335 accuracy is impressive
7 when configured for high values of IPEAK. When configured with low values
8 of IPEAK, however, there can be up to 40% error.
9 
10 This file contains the typical error for all IPEAK, VOUT, and battery voltage
11 conditions encoded as a signed 16bit integer. An array of these errors will
12 build into the firmware as LTC3335_Software_Correction_Table[] for the IPEAK
13 and VOUT values specified in LTC3335_Config.h. The index into this table
14 will be the battery voltage, which may vary over life of a product using the
15 LTC3335.
16 
17 To use these correction factors to adjust the coulomb count and battery current
18 measurement, multiply the unadjusted value by the correction factor and shift
19 right by 16. For example:
20 
21  If IPEAK = 10mA (LTC3335_IPEAK_CONFIGURATION == LTC3335_IPEAK_CONFIGURATION_10MA),
22  VOUT = 3.3V (LTC3335_OUTPUT_VOLTAGE == LTC3335_OUTPUT_VOLTAGE_3_3V), and
23  VBAT = 4.5V (vbat = 4500mV), then the LTC3335_Get_Software_Correction_Factor(uint16_t vbat)
24  function will return 8826 as the software correction factor. This value translates to
25  8826/65536 = 13.4%, which matches the error shown in G43 of the datasheet.
26 
27  If a prescaler of 5 was selected for this example, then one count of the
28  accumulator should represent 31638 mAs (0.0088 Ah) of charge. To adjust for
29  the expected error, we should add the fraction returned by the
30  LTC3335_Get_Software_Correction_Factor(uint16_t vbat) function to the
31  unadjusted coulomb count:
32 
33  31638mAs + (31638mAs * 8826) >> 16 = 35899 mAs.
34 
35 Note! - These correction factors are only valid at room temperature and for the recommended
36 Coilcraft LPS5030-xxxMRB inductor listed in the datasheet. Contact LTC applications if
37 interested in generating the software correction factors for other tempeature conditions
38 and inductors.
39 
40 @verbatim
41 
42 The LTCÂ®3335 is a high efficiency, low quiescent current
43 (680nA) buck-boost DC/DC converter with an integrated
44 precision coulomb counter which monitors accumulated
45 battery discharge in long life battery powered applications.
46 The buck-boost can operate down to 1.8V on its input and
47 provides eight pin-selectable output voltages with up to
48 50mA of output current.
49 
50 The coulomb counter stores the accumulated battery discharge
51 in an internal register accessible via an I2C interface.
52 The LTC3335 features a programmable discharge alarm
53 threshold. When the threshold is reached, an interrupt is
54 generated at the IRQ pin.
55 
56 To accommodate a wide range of battery types and sizes,
57 the peak input current can be selected from as low as 5mA
58 to as high as 250mA and the full-scale coulomb counter
59 has a programmable range of 32,768:1.

354 //! Returns the software correction factor for a specified LTC3335_IPEAK_CONFIGURATION, LTC3335_OUTPUT_VOLTAGE, and battery voltage.
355 //! Note! - These corrections factors are only valid at room temperature and for the recommended Coilcraft LPS5030-xxxMRB inductor.
356 //! @return TRUE if the LTC3335 communication was successful.
357 int16_t LTC3335_Get_Software_Correction_Factor(uint16_t vbat)
358 {
359  if (vbat <= LTC3335_VBAT_MIN)
360  {
361  return pgm_read_word(&LTC3335_Software_Correction_Table[LTC3335_VBAT_TO_TABLE_INDEX(LTC3335_VBAT_MIN)]);
362  }
363  else if (vbat >= LTC3335_VBAT_MAX)
364  {
365  return pgm_read_word(&LTC3335_Software_Correction_Table[LTC3335_VBAT_TO_TABLE_INDEX(LTC3335_VBAT_MAX)]);
366  }
367  else
368  {
369  uint8_t index = LTC3335_VBAT_TO_TABLE_INDEX(vbat);
370  int16_t factor1 = pgm_read_word(&LTC3335_Software_Correction_Table[index]);
371  int16_t factor2 = pgm_read_word(&LTC3335_Software_Correction_Table[index + 1]);
372  return factor1 + ((int32_t) (factor2 - factor1) * (vbat - (LTC3335_VBAT_MIN + index * LTC3335_VBAT_STEP)) + LTC3335_VBAT_STEP/2) / (LTC3335_VBAT_STEP);
373  }
374 }
