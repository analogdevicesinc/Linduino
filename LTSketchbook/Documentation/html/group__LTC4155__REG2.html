Detailed Description

REG2 Register.

7:4 3:2 1:0
ICHARGE VFLOAT CXSET
  • CommandCode: 0x02
  • Contains Bit Fields:
    • ICHARGE : Battery charger current limit setting.
    • VFLOAT : Battery charge voltage setting.
    • CXSET : End-of-Charge indication current threshold setting. This threshold can also be used for charge termination if TIMER is set to 8_Hour_or_CoverX.
