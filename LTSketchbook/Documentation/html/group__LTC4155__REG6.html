Detailed Description

REG6 Register.

7 6 5 4 3 2 1 0
ENABLE_CHARGER_INT ENABLE_FAULT_INT ENABLE_EXTPWR_INT ENABLE_OTG_INT ENABLE_AT_ILIM_INT ENABLE_INPUT_UVCL_INT REQUEST_OTG RESERVED

Interrupt Mask Resister

  • CommandCode: 0x06
  • Contains Bit Fields:
    • ENABLE_CHARGER_INT : Enable battery CHARGER_STATUS interrupts.
    • ENABLE_FAULT_INT : Enable OVP_ACTIVE, OTG_FAULT, BAD_CELL, and NTCSTAT = Hot_Fault interrupts.
    • ENABLE_EXTPWR_INT : Enable USBSNS_GOOD, WALLSNS_GOOD, EXT_PWR_GOOD external power available interrupts.
    • ENABLE_OTG_INT : Enable ID_PIN_DETECT and OTG_ENABLED USB On-The-Go interrupts.
    • ENABLE_AT_ILIM_INT : Enable AT_INPUT_ILIM interrupts.
    • ENABLE_INPUT_UVCL_INT : Enable INPUT_UVCL_ACTIVE interrupts.
    • REQUEST_OTG : USB On-The-Go step-up voltage converter manual activation setting.
    • RESERVED : Always set to 0.
